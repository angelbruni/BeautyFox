What's new in BeautyFox Release Candidate 3 - 1.0

  • Changed prefs;
  • Small changes to localisation strings;
  • Localisation now uses Firefox's system for the most part;
  • Created functions for easily reading and writing to different types of prefs;
  • Created function for easily reading registry;

  • Separeted a bunch of code into its own script files again;
  • Completely removed previous settings code in favour of the brand new SettingsManager;

    • Combobox;
    • Commandlinkbutton;

    • Navigation buttons;
    • Preview container;
    • Tile;
    • Usertile.

    • The ZuneMetro framework.
    • All the previously known settings +
      • Amount of popular site rows in new tab page;
      • DWMBlurGlass accent colour for the UI;
      • Custom colour for the UI that is completely custom and does not rely in accent colour.
    • Setting previews with images, descriptions and warnings;
    • Very detailed about tab with information about:
