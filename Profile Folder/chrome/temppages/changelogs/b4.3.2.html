What's new in BeautyFox Beta 4.3.2

  • "tr" localisation by Brawllux;
  • Fixed layout guide glow not masking;
  • Added function for calling ShellExecuteW (removed networkDiagnostiscs.uc.js and windowsUpdate.uc.js files);
  • Microsoft Edge tab button for IE11Win10 with options for hiding or using old or new Edge icon;
  • Made certain additional settings only appear depending on which appearance was chosen previously to clean up the screen;
  • Fixed statusbar getting cutoff if window is too small;
  • Fixed dragging tabs without tabsOnNavRow being weird;
  • Fixed new tab button not having hover effect and an icon on IE11Win10 if tabs are overflowing;
  • Wizard no longer gets confused on what to choose by default in comboboxes if keys are missing from about:config;
  • Fixed AWM option becoming available at times even when AWM is not found;
  • Fixed aboutDialog not displaying "64-bit" after pressing shift;
  • Internet options now uses the ShellExecuteW call from the function.
